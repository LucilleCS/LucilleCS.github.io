CSCE 242 : Web Applications

Lucille O'Shea

In web applications we will be using html, css and javascript to design both the front and end of websites.

The class also teaches the server side and teaches to store and retieve data.

This specific class is on tuesdays and thursdays from 8:30-9:45 am.

Assignments:

Assignment 1: Basic HTML

Using basic and beginner html, made a website
that had headings, paragraphs, images, tables, and links.

Assignment 2: Basic CSS

Made a webpage, focused on using CSS styling.
things such as nagivation, color schemes, and color borders
are used here.

Assignment 3: Page Layout

using flexbox and a main focus, I made
a website that focuses on the setup and design.
and utilizing @media to resize appropriately.
